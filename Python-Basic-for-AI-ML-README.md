## 🚀 Getting Started

### Prerequisites

- Python `3.8+`
- Jupyter Notebook or JupyterLab *(recommended)*

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/Abhishek-Maurya-D/Python-Basic-for-AI-ML.git
cd Python-Basic-for-AI-ML

# 2. (Optional) Create a virtual environment
python -m venv venv
source venv/bin/activate      # macOS/Linux
venv\Scripts\activate         # Windows

# 3. Install Jupyter
pip install jupyter

# 4. Launch notebooks
jupyter notebook
```

Then navigate to any folder and open the `.ipynb` files — start with `Python_Fundamental_1` if you're beginning from scratch.
